# 生日网站重设计 - 设计文档

**日期**: 2026-05-11
**状态**: 已确认

## 概述

将现有两个独立页面（index.html 非生日页 + index.md 生日页）合并为一个智能单页应用。根据本地日期自动切换生日模式（6月26日）和非生日模式。采用游戏化互动风格，生日模式为关卡式揭秘+彩蛋收集，非生日模式为恶搞趣味+迷你互动。

## 架构

- **单文件 HTML**（index.html），包含全部 CSS/JS
- **日期检测模块**：读取本地日期（`new Date()`），判断 month=6 且 day=26，自动切换模式
- **状态机**：`idle → stage1 → stage2 → stage3 → finale`（生日模式）
- **特效引擎**：Canvas 粒子系统、屏幕震动、闪光效果
- **音效系统**：Web Audio API 合成音效，无需外部文件
- **彩蛋追踪**：收集数存入 LocalStorage
- **访客计数**：非生日模式用，LocalStorage 记录

## 配色

### 生日模式
- 主背景：深蓝紫渐变星空（Canvas 粒子背景）
- 强调色：金色 `#FFD700`、珊瑚粉 `#FF6B6B`
- UI 元素：圆角卡片 + 柔和发光

### 非生日模式
- 主背景：暗黑 + 月光蓝
- 强调色：霓虹绿 `#00FF88`、冷蓝 `#4ECDC4`
- UI 元素：棱角分明 + 像素风边框

## 字体
- 标题：游戏风粗体 fallback 系统粗体
- 正文：`Comic Neue`, cursive（趣味感）

## 游戏化 UI 特征
- HP 条样式进度条
- 像素风按钮带按压动画
- 飘分效果（"+1 🥚" 浮动文字）
- 关键时刻屏幕震动

## 音效（Web Audio API 合成）
- 按钮点击：短促 "pop"
- 气球爆裂：低频 "boom"
- 烟花：高频 "shimmer"
- 彩蛋收集：清脆 "ding"

## 生日模式（6月26日）

### 阶段1：拆礼物
- 屏幕中央 CSS 3D 翻转礼物盒（金色丝带）
- 背景：Canvas 星空粒子场
- 点击 → 盒盖弹开 + 粒子爆发
- 金色游乐园门票卡片飞入，显示：
  > 🎪 地球人的生日游乐园 · 🎫 凭此票入场 · 📅 6月26日
- 同时屏幕闪现隐藏彩蛋（需快速点击收集，至少3个才能解锁隐藏结局）

### 阶段2：游乐园（3个摊位，顶部进度条 🎈→🕯️→🎂→🎆）

**摊位1 — 戳气球：**
- 10+ 彩色气球飘浮，点击戳破
- 每个气球爆开有粒子效果 + 祝福词
- 全部戳完解锁下一个

**摊位2 — 吹蜡烛：**
- 蛋糕中央蜡烛火焰 CSS 动画摇曳
- 吹气（Web Audio API 麦克风音量检测）或连续点击吹灭
- 不支持麦克风则手指滑动/快速点击替代

**摊位3 — 切蛋糕：**
- 刀跟随鼠标/手指，划一刀切开蛋糕
- 内部爆出彩带 + 隐藏彩蛋
- 切完后弹出祝福

### 阶段3：惊喜绽放
1. 屏幕震动 0.5s
2. Canvas 烟花秀 5 秒
3. "生日快乐，地球人！" 金色大字飞入
4. 展示收集的彩蛋数（N/5）
5. 集齐5个 → 隐藏结局（聚光灯+专属祝福+再来一次/分享按钮）
6. 尾声："🎪 游乐园打烊了，明年再见！"

## 非生日模式（其他364天）

### 首次访问
- 发光"锁着的门"图标
- 打字机文字："你来早了...今天不是 6月26日"

### 搞笑对话轮播
- 多条搞笑消息点击切换

### 假礼物小游戏
- 礼物盒屏幕乱跑，点击跳跃
- 每次点击显示搞笑文字（"差一点！""想得美！"等）
- 点击20次解锁安慰奖

### 倒计时
- 距离下一个6月26日的倒计时

### 访客计数
- LocalStorage 记录："你是第N个提前来的人"

---

## 技术要点
- 纯 HTML/CSS/JS 单文件
- Canvas 用于粒子系统+烟花
- Web Audio API 用于音效合成
- LocalStorage 用于彩蛋计数和访客计数
- 响应式适配移动端
- 所有效果纯代码实现，无外部依赖
